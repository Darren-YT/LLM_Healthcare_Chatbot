# LLM Healthcare Chatbot

This project was developed as part of COMP293: AI in Healthcare.

The project focuses on building a healthcare question-answering chatbot using Large Language Models (LLMs), custom healthcare datasets, and LoRA fine-tuning techniques.

## Technologies

* Python
* LoRA Fine-Tuning
* Qwen LLM
* JSONL Datasets
* Pandas

## Features

* Healthcare question-answer dataset preparation
* Dataset cleaning and preprocessing
* CSV to JSONL conversion
* LLM fine-tuning workflow
* Chatbot inference and evaluation

## Project Structure

* `chatbot.ipynb` – Main notebook for training and evaluation
* `combine.py` – Dataset processing and merging
* `readfile.py` – Data loading utilities
* `combined_chatbot_dataset.csv` – Healthcare Q&A dataset
